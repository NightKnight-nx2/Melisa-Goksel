# 💜 Melisa Göksel — Sürpriz Doğum Günü Sitesi

Bu projeyi kız arkadaşım için yaptım. Eğer siz de sevdiğiniz birine böyle bir sürpriz yapmak isterseniz, bu projeyi istediğiniz gibi kullanabilirsiniz. İçerikleri kendi hikayenize göre düzenleyip GitHub Pages üzerinden ücretsiz yayınlayabilirsiniz.

## Proje Hakkında

Tek sayfalık, tamamen scroll animasyonlu bir web sitesi. Her bölüm kaydırdıkça hayat buluyor — papatyalar açılıyor, gezegenler dönüyor, karakterler sahneye giriyor.

**Bölümler:**
1. **Hero** — İsim ve papatya animasyonları
2. **Uzay** — Jüpiter + Plüton orbit sistemi, yıldız kayan yıldızlar
3. **Bikini Kasabası** — Sualtı efekti, denizanalar, köpük geçişi
4. **Madagascar** — Marty sahneye giriyor
5. **Cars** — Lightning McQueen piste iniyor
6. **Tatlı Krizi** — Kek, macaron, çilek, dondurma
7. **Özel Anlar** — Birlikte yaşanan özel günler
8. **Doğum Günü** — Konfeti, yıldızlar, kutlama

## Kullanılan Teknolojiler

- **GSAP 3 + ScrollTrigger** — Tüm scroll animasyonları çift yönlü (yukarı kaydırınca geri döner)
- **Vanilla HTML / CSS / JS** — Çerçeve yok, bağımlılık yok
- **GitHub Pages** uyumlu — Olduğu gibi yayınlanabilir

## Nasıl Kullanılır?

1. Bu repoyu fork'layın veya klonlayın
2. `index.html` içindeki isimleri, tarihleri ve alıntıları kendi hikayenize göre düzenleyin
3. `assets/` klasörüne kendi görsellerinizi ekleyin
4. GitHub Pages'i etkinleştirin (Settings → Pages → main branch / root)
